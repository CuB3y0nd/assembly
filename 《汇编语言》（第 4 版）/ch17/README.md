## 检测点 17.1

```
“在 int 16h 中断例程中，一定有设置 IF=1 的指令。” 这种说法正确吗？
```

- 正确。只有响应键盘输入的外中断后才可以向键盘缓冲区写入数据，由此 `int 16h` 中断才得以读取到输入。故 `IF` 位必须设置为 `1`。
